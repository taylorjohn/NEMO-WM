What It Can't Do (Yet)

Navigate real kitchens. Speak thousands of words. Recognize faces. Manipulate objects with hands. The architecture handles all of these — the scale doesn't, yet. The claim isn't that we built a human. The claim is that we built every subsystem a human uses to think, wired them together, and proved they work.
